Тестовое задание 
 Написать погодное iOS-приложение на Swift с использованием UIKit, полностью программно без Storyboard. 
 UI - Отобразить один экран с погодной информацией: текущая, почасовая (показывать оставшиеся часы из текущего дня и все часы следующего), прогноз погоды на 3 дня. - Обработать показ загрузки и ошибку, если что-то пошло не так, с кнопкой повторного запроса
